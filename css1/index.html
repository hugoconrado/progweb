Software Livre

O software livre preza pela liberdade dos usuários. Para que a
liberdade do usuário seja preservada, ele deve ter quatro poderes
em relação ao software que ele usa:

  • Ler o código dos programas que utiliza e estudá-los;
  • Distribuir livremente o programa para outros usuários;
  • Modificar o código para atender às suas necessidades;
  • Distribuir livremente a versão modificada do código;
